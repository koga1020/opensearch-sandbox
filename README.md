# OpenSearch Sandbox

OpenSearchのキャッチアップのための素振り環境

## open dashboard

```
docker compose up -d
open http://localhost:5601
```

## run hono dev server

```bash
npm ci
npm run dev
open http://localhost:3000
```

## note

[tutorial_directory](http://localhost:5601/app/home#/tutorial_directory)からサンプルデータを挿入でき、レコード操作を試せる
